# Cluely Site - Лендинг продукта

Современный лендинг для Cluely - невидимого AI-ассистента для встреч и интервью.

## Технологии

- **Next.js 14** с App Router
- **TypeScript**
- **Tailwind CSS** для стилизации
- **Static Site Generation (SSG)** для SEO-оптимизации
- Полностью серверный рендеринг

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка для продакшена

```bash
npm run build
```

Собранный сайт будет в папке `out/` и готов к деплою на любой статический хостинг.

## Структура проекта

```
cluely-site/
├── app/
│   ├── layout.tsx      # Основной layout с мета-тегами
│   ├── page.tsx        # Главная страница
│   └── globals.css     # Глобальные стили
├── components/
│   ├── Hero.tsx        # Hero секция
│   ├── Features.tsx    # Возможности продукта
│   ├── Benefits.tsx    # Преимущества
│   ├── Pricing.tsx     # Цены
│   ├── CTA.tsx         # Призыв к действию
│   └── Footer.tsx      # Футер
└── public/             # Статические файлы
```

## SEO-оптимизация

- Все страницы используют SSG (Static Site Generation)
- Оптимизированные мета-теги для поисковых систем
- Семантическая HTML разметка
- Оптимизированные изображения (готово к добавлению)

## Деплой

Сайт можно задеплоить на:
- Vercel (рекомендуется для Next.js)
- Netlify
- GitHub Pages
- Любой статический хостинг

Просто загрузите папку `out/` после сборки.
