# Блог Cluely

Страница блога создана с использованием Next.js SSG для SEO-оптимизации.

## Структура

- `/blog` - список всех постов
- `/blog/[id]` - отдельная страница поста

## Добавление новых постов

Чтобы добавить новый пост, нужно:

1. Добавить объект поста в массив `blogPosts` в файле `app/blog/page.tsx`
2. Добавить тот же объект в объект `blogPosts` в файле `app/blog/[id]/page.tsx`
3. При необходимости добавить изображение в `public/images/blog/`

## Структура поста

```typescript
{
  id: number,                    // Уникальный ID
  title: string,                  // Заголовок
  date: string,                  // Дата в формате "DD MMMM YYYY"
  author: string,                // Автор
  excerpt: string,                // Краткое описание (анонс)
  content: string,                // Полный текст статьи
  category: string,               // Категория: 'Обновления', 'Функции', 'Улучшения'
  image: string,                  // Путь к изображению (например '/images/blog/name.jpg')
  tags: string[],                 // Массив тегов
}
```

## Изображения

Изображения размещаются в `public/images/blog/`. Рекомендуемые размеры:
- Главное изображение: 1200x630px
- Миниатюра: 800x450px

## Фичи из постов

Посты основаны на реальных функциях из репозитория:
- Поддержка Ollama (локальный AI)
- Умный анализ скриншотов с Gemini 2.0 Flash
- Аудио-анализ и транскрипция
- Контекстуальный чат с памятью
- Кроссплатформенная поддержка
- Прозрачный интерфейс

Все посты используют SSG для максимального SEO.
